# Modern SQL Data Warehouse Project

## Overview

This project demonstrates the design and implementation of a Modern Data Warehouse using SQL Server. The solution follows the Medallion Architecture (Bronze, Silver, and Gold layers) to ingest, clean, transform, and model data from multiple source systems for analytics and reporting.

## Project Objectives

* Build an end-to-end Data Warehouse solution.
* Develop ETL pipelines for data ingestion and transformation.
* Implement data quality checks and cleansing processes.
* Create a Star Schema dimensional model.
* Prepare analytics-ready datasets for Power BI dashboards.

## Architecture

Bronze Layer → Raw data ingestion from CSV files

Silver Layer → Data cleansing, standardization, and transformation

Gold Layer → Business-ready dimensional models and fact tables

## Technologies Used

* SQL Server
* SQL (Stored Procedures, Views, Window Functions)
* ETL Processes
* Data Warehousing
* Star Schema Modeling
* Git & GitHub
* Power BI

## Dataset

The project uses customer, product, sales, and location datasets simulating data from CRM and ERP systems.

## Key Features

* Automated data loading using SQL scripts
* Data cleansing and validation
* Duplicate removal and null handling
* Fact and Dimension table creation
* Analytics-ready data model
* Documentation and version control

## Skills Demonstrated

SQL Development, ETL, Data Warehousing, Data Modeling, Data Quality Management, Business Intelligence, and Data Analytics.
